# faircontract.github.io
How to add own token on: [FairContracts](https://speedypy.github.io/faircontracts)

Fork the Repository,
Edit Tokens.json file some as.

```
Token#1 {
        Token Name: "name",
        Token Symbol: "symbol",
        Token Website: "link",
        Token Group Link: "tg",
};
```

There is no fee required for listing...
